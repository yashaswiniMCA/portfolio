Project

Stress Detection Using Machine Learning

Developed a machine learning system for detecting stress in IT employees using Convolutional Neural Networks (CNNs). The CNN model, trained with a training accuracy of 87.34% and validated with an accuracy of 98.45%, analyzes physiological data represented as images. The system supports two modes: uploading images or live camera feed for real-time stress detection. Integrated with secure company and employee login for personalized monitoring and access control. This solution aids in managing employee well-being by providing accurate and immediate stress assessments.

Project Objective: Develop a machine learning model to detect stress in IT employees. Enable early detection to aid in mental health and productivity.

Technology Stack: Machine Learning Model: Convolutional Neural Networks (CNNs). Programming Language: Python for model development. Data Processing and Visualization: OpenCV for image processing and real-time video analysis.

Data Source and Preparation: Collected data based on physiological cues (e.g., facial expressions, eye patterns). Preprocessed data for accurate image classification.

Modes of Detection: Image Upload: Employees can upload a photo, which is analyzed to detect stress levels. Live Camera Feed: Uses live video feed for real-time stress detection, providing instant results.

System Features: Accuracy: Achieved 87.34% training accuracy and 98.45% validation accuracy. Secure Login: Individualized employee and admin login for secure data management. Data Privacy: Encrypted data storage to maintain privacy.

Output and Analysis: Displays stress levels visually with real-time feedback on detection. Provides detailed reports on detected stress patterns to HR and managers.

Benefits: Helps organizations monitor employee well-being. Supports early intervention to reduce burnout and improve productivity.
